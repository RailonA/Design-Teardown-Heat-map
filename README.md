# Design-Teardown-Heat-map

A grayscale heat map of the Smashing Magazine (http://smashingmagazine.com/) website that indicates which elements have the greatest weight in the visual hierarchy.



![screenshot](./asset/)
 
## Built With

- HTML5
- CSS
- Font Awesome
